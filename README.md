# 犬の飼育適性診断アプリ

犬を飼いたい方の生活環境や希望に基づいて、最適な犬種を提案するWebアプリケーションです。
見た目やイメージだけでなく、実際の飼育条件を考慮することで、ミスマッチを防ぎ、適切なマッチングを実現します。

## 🔗 デモ
**https://dog-matching-app.vercel.app/**

## 📱 機能
- **質問形式の診断**: 住居形態、散歩時間、留守時間など15項目の質問に回答
- **最適な犬種提案**: ユーザーの回答に基づき、5種の犬種をおすすめ度付きで提示
- **詳細な情報提供**: 各犬種について、向いている理由・注意点・飼育時の注意点などを表示
- **💰 経済面の可視化**: 犬種ごとの年間ランニングコスト（餌代、医療費、トリミング代等）の概算を表示
- **🏥 健康管理情報**: 犬種ごとにかかりやすい病気・怪我とその予防策・対策を提示
- **レスポンシブデザイン**: スマートフォンからでも快適に利用可能

## 🎯 開発の背景
犬を飼いたいが「自分に合う犬種が分からない」というユーザーの不安を解消することを目的に開発しました。
見た目やイメージだけで犬種を選んでしまうことによるミスマッチを防ぎ、飼育放棄や衝動買いを減らすことを目指しています。

### 解決する課題
- 見た目・流行だけで犬種を選んでしまう問題
- 運動量やしつけの難易度を理解していないことによるギャップ
- 犬種情報が断片的で比較しづらい状況

## 🌟 他サービスとの差別化ポイント
既存の犬種診断サイトとの主な違い：

1. **経済的負担の可視化**
   - 単なる犬種紹介にとどまらず、年間ランニングコストを具体的に提示
   - 餌代、医療費、トリミング代などの項目別概算により、飼育の現実的なコストを把握可能
   - 「飼いたい」という感情だけでなく、経済的な準備ができるかを判断する材料を提供

2. **予防医療の視点**
   - 各犬種がかかりやすい病気・怪我を事前に提示
   - 具体的な予防策・対策を記載することで、将来のトラブルに備えられる
   - ペット保険の検討材料にもなる情報提供

## 🛠️ 使用技術
- **フロントエンド**: Next.js, React
- **AI**: LLM（大規模言語モデル）による診断ロジック
- **デプロイ**: Vercel
- **情報源**: 環境省、日本獣医師会、JKC（ジャパンケネルクラブ）などの信頼できるデータ

## 💡 工夫した点
1. **経済面と健康面の実用情報**: 他の診断サイトにはない、年間コストや病気リスクを提示することで、より現実的な飼育判断をサポート
2. **Claude AIの効果的活用**: 短期間での開発を実現し、複雑な診断ロジックや多様な犬種データの整理を効率化
3. **ユーザー目線の質問設計**: 専門用語を避け、誰でも答えやすい質問項目を設定
4. **安全性への配慮**: 「絶対」などの断定表現を避け、「傾向として」「一般的に」といった表現を使用
5. **公平性の確保**: 特定の犬種への偏見や差別的な表現を排除
6. **情報の信頼性**: 公的機関や専門団体のデータを参照し、根拠のある情報を提供

## 📊 評価基準
開発にあたって以下の品質基準を設定しました：
- **正確性**: 一般的な犬種特性に基づいた情報提供
- **安全性**: 危険・虐待につながる内容の排除
- **一貫性**: 回答内で矛盾のない診断結果
- **簡潔性**: 専門用語を最小限に抑えた分かりやすい説明
